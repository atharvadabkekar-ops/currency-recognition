# Indian Currency Note Classification

This project uses a pretrained ResNet‑50 model fine‑tuned with PyTorch
to classify Indian currency denominations using images.

## Features
- GPU training (RTX 4050)
- Modular architecture
- Transfer learning
- Real‑time inference support

## Dataset
Images are organized by denomination folders.

## Model
ResNet‑50 pretrained on ImageNet.
